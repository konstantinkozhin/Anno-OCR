# Anno OCR: инструкция пользователя

## 1. Что это за программа

`Anno OCR` — приложение для распознавания и разметки текста на изображениях (в том числе архивных документов).

Программа позволяет:
- находить текстовые области;
- распознавать текст;
- исправлять текст;
- сортировать слова в строки и блоки;
- вручную править разметку;
- экспортировать результат в `JSON`, `COCO`, `PDF`.

## 2. Быстрый старт

1. Откройте папку с изображениями: `Ctrl+O`.
2. Переключайтесь между страницами: `←` и `→`.
3. Запустите OCR:
- `1` — детекция;
- `2` — распознавание;
- `3` — коррекция;
- `4` — сортировка;
- `5` — полный pipeline.
4. Проверьте разметку справа (слова/строки/блоки), при необходимости исправьте вручную.
5. Экспортируйте результат через меню `Импорт/экспорт`.

## 3. Основные элементы интерфейса

## 3.1 Верхнее меню

- `Файл` — открыть папку, окно «О программе», выход.
- `Настройки` — параметры приложения.
- `Импорт/экспорт` — импорт и экспорт данных.
- `Предпросмотр` — просмотр итогового текста.

## 3.2 Верхняя панель

- навигация по страницам;
- масштаб;
- кнопка приглушения фона;
- кнопка показа/скрытия расшифровок;
- поворот страницы;
- обновление страницы (`F5`).

## 3.3 Рабочая область

- слева/в центре — изображение (canvas);
- справа — структура: блоки, строки, слова.

## 4. Режимы редактирования на canvas

- `Box` (`Q` / `Й`) — прямоугольные боксы.
- `Polygon` (`A` / `Ф`) — полигональные боксы.
- `Table` (`Z` / `Я`) — режим таблиц.

## 5. Ручная разметка и правка

## 5.1 Выделение и редактирование

- Клик по боксу — выбрать.
- Перетаскивание — перемещение бокса.
- Тяните за маркеры — изменение размера.
- Двойной клик по слову — редактирование текста.

## 5.2 Мультивыделение

- `Ctrl+клик` — добавить/убрать слово из выделения.
- `Ctrl+A` — выбрать все слова (вне режима таблицы).

## 5.3 Действия с выделением

- `Enter` — создать строку из выбранных слов.
- `Space` — объединить выбранные слова в один бокс.
- `Delete` / `Backspace` — удалить выбранное.

## 5.4 Буфер обмена разметки

- `Ctrl+X` — вырезать выбранные слова/строки.
- `Ctrl+V` — вставить.

## 5.5 Ускоренный режим Shift-цепочки

Если нужно быстро собирать последовательность слов в строку:

1. Выберите бокс-якорь.
2. Удерживайте `Shift`.
3. Либо рисуйте новые боксы, либо кликайте по существующим.
4. Каждый следующий бокс автоматически ставится после предыдущего в строке якоря.

## 6. Работа с таблицами

В режиме `Table`:
- можно рисовать и править сетку вручную;
- можно запускать автоопределение таблицы по 4 углам;
- можно менять число строк/столбцов в редакторе таблицы.

## 7. Импорт и экспорт

Окно `Импорт/экспорт` поддерживает:

- Импорт: `PDF`, `JSON`, `COCO`.
- Экспорт: `PDF`, `JSON`, `COCO`.
- Отдельно: скачивание аудит-отчета.

Для `JSON` и `COCO` рядом может создаваться файл с таблицами (`*_tables.json`), который автоматически учитывается при импорте.

## 8. История действий

- `Ctrl+Z` — отменить.
- `Ctrl+Y` или `Ctrl+Shift+Z` — вернуть.

Отмена/повтор работают и для поворота страницы, и для изменений разметки.

## 9. Горячие клавиши

- `Ctrl+O` — открыть папку.
- `Ctrl+Q` — выход.
- `←` / `→` — предыдущая / следующая страница.
- `F5` — обновить страницу.
- `Ctrl+Z` — Undo.
- `Ctrl+Y` или `Ctrl+Shift+Z` — Redo.
- `Ctrl+X` / `Ctrl+V` — вырезать / вставить.
- `Ctrl+A` — выделить все слова.
- `Delete` / `Backspace` — удалить выделенное.
- `Enter` — создать строку или редактировать активное слово.
- `Space` — объединить выбранные слова.
- `Ctrl++`, `Ctrl+-`, `Ctrl+0` — масштаб +/−/сброс.
- `1`, `2`, `3`, `4`, `5` — OCR-операции.
- `L/Д` — приглушить фон.
- `E/У` — показать/скрыть расшифровки.
- `V/М` — переключить компоновку панели.
- `R/К` — повернуть страницу.
- `Q/Й`, `A/Ф`, `Z/Я` — режимы Box/Polygon/Table.
