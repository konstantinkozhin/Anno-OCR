## О программе

**Anno OCR** — инструмент для работы с рукописными и историческими документами. Программа позволяет:

- **Распознавать текст** в изображениях документов — локально или через API
- **Размечать** области текста вручную или автоматически
- **Организовывать** распознанные слова в строки и блоки с правильным порядком чтения
- **Корректировать** результаты распознавания с помощью языковой модели (включая дореформенную орфографию)
- **Экспортировать** результаты в JSON, COCO-формат или PDF

Программа распространяется в виде одного `.exe` файла — ничего дополнительно устанавливать не нужно.

## Установка и запуск

1. Перейдите на страницу [**Releases**](https://github.com/konstantinkozhin/anno-ocr/releases/latest) или [**Яндекс Диск**](https://disk.yandex.ru/d/Hc4lDQJKahEbIg)
2. Скачайте файл `Anno_OCR.exe`
3. Запустите — готово!

> **Системные требования:** Windows 10/11, 64-bit, более 4 ГБ оперативной памяти.  
> При первом использовании локального OCR модели скачиваются автоматически в `~/.manuscript/weights/`.

### Холст (центр)

Холст — основное рабочее пространство. На нём отображается изображение документа и все аннотации:

**Создание нового бокса:**  
Нажмите и потяните левой кнопкой мыши по пустой области — появится зелёный пунктирный прямоугольник. Отпустите — бокс создан.

**Жёлтые стрелки:**  
Между последовательными словами выбранной строки рисуются стрелки, показывающие порядок чтения.

---

### Панель структуры (справа)

Иерархическое представление аннотаций текущей страницы:

| Секция | Содержимое |
|--------|------------|
| **Блоки** | Именованные блоки текста. Каждый раскрывается и содержит строки, а строки — слова |
| **Без блока** | Строки, не назначенные ни одному блоку |
| **Свободные слова** | Слова, не назначенные ни одной строке |

**Что можно делать:**

- **Клик по блоку** — выделить блок, на холсте замигают все его слова
- **Двойной клик по блоку** — переименовать
- **Клик по строке** — выделить строку, на холсте замигают её слова
- **Клик по слову** — выделить слово, холст центрируется на нём
- **Двойной клик по слову** — редактировать текст
- **Перетаскивание слов** — переместить между строками или в свободные слова
- **Перетаскивание строк** — переместить между блоками или в «без блока»
- **Ctrl+клик** — множественное выделение строк или слов

## Сценарии использования

### Быстрый старт — от изображения до текста

> Самый простой путь: открыть папку с изображениями и запустить полный пайплайн.

**Шаг 1. Откройте папку с изображениями**

`Файл → Открыть папку` или `Ctrl+O`. Выберите папку с изображениями документов (JPG, PNG, BMP).

**Шаг 2. Запустите пайплайн**

Нажмите кнопку **Пайплайн** на боковой панели (или клавишу `5`). Программа автоматически:
1. Найдёт области текста
2. Распознает текст в каждой области
3. Исправит ошибки языковой моделью
4. Отсортирует элементы

**Шаг 3. Просмотрите результат**

На холсте появятся цветные боксы с распознанным текстом. В панели структуры справа — иерархия блоков, строк и слов.

**Шаг 4. Скопируйте или экспортируйте текст**

- Нажмите **Предпросмотр** в меню — откроется окно с полным текстом. Кнопка «Копировать всё» скопирует текст в буфер обмена.
- Или экспортируйте через `Файл → Экспорт PDF` / `Экспорт JSON`.

---

### Работа с PDF

1. `Файл → Импорт PDF` — выберите PDF-файл
2. Каждая страница будет сохранена как PNG (200 DPI) в новую папку
3. Папка откроется автоматически — можно начинать распознавание

**Экспорт в PDF:**  
`Файл → Экспорт PDF` — создаёт PDF, где на каждой странице слева — оригинальное изображение, справа — распознанный текст.

---

### Ручная разметка и корректировка

Если автоматическое распознавание допустило ошибки, можно исправить вручную:

| Действие | Как сделать |
|----------|-------------|
| **Создать бокс** | Нарисуйте прямоугольник на холсте (зажмите ЛКМ и потяните) |
| **Изменить размер бокса** | Выделите бокс → потяните за зелёные угловые ручки |
| **Переместить бокс** | Выделите бокс → кликните ещё раз и перетащите |
| **Редактировать текст** | Дважды кликните по боксу на холсте или по слову в панели |
| **Удалить бокс** | Выделите бокс → `Delete` |
| **Объединить слова в строку** | Выделите несколько боксов (`Ctrl+клик` или зажмите ПКМ) → `Enter` |
| **Отменить последнее действие** | `Ctrl+Z` |

### Пакетная обработка

<img width="757" height="637" alt="image" src="https://github.com/user-attachments/assets/5fcdad4f-9f0a-4f25-afdf-bb780274adaf" />

Для обработки множества страниц сразу:

1. **Правой кнопкой** мыши нажмите на любую OCR-кнопку (Детекция, Распознавание и т.д.)
2. Откроется окно **пакетной обработки** с миниатюрами всех изображений
3. Выберите нужные страницы (или `Ctrl+A` для всех)
4. Нажмите **Старт** — обработка идёт параллельно в несколько потоков
5. Прогресс отображается на шкале, можно отменить в любой момент

### Экспорт результатов

| Формат | Меню | Описание |
|--------|------|----------|
| **JSON (датасет)** | `Файл → Экспорт датасета (JSON)` | Все страницы с блоками, строками, словами, координатами в пикселях |
| **COCO JSON** | `Файл → Экспорт COCO (JSON)` | Стандартный COCO-формат аннотаций для ML-задач |
| **PDF** | `Файл → Экспорт PDF` | Двухколоночный PDF: изображение + распознанный текст |

Все форматы можно также **импортировать** обратно через соответствующие пункты меню.

## Горячие клавиши

### Навигация

| Клавиша | Действие |
|---------|----------|
| `←` / `→` | Предыдущее / следующее изображение |

### OCR-операции

| Клавиша | Действие |
|---------|----------|
| `1` | Детекция (найти области текста) |
| `2` | Распознавание |
| `3` | Коррекция |
| `4` | Сортировка |
| `5` | Полный пайплайн |

> Работают только когда фокус на холсте.

### Редактирование

| Клавиша | Действие |
|---------|----------|
| `Ctrl+Z` | Отменить последнее действие |
| `Ctrl+X` | Вырезать выделенные слова/строки |
| `Ctrl+V` | Вставить слова в строку / строки в блок |
| `Ctrl+A` | Выделить все боксы на странице |
| `Delete` / `Backspace` | Удалить выделенные элементы |
| `Enter` | Создать строку из мульти-выделенных боксов / редактировать текст одного бокса |
| `Escape` | Сбросить выделение |

### Масштаб

| Клавиша | Действие |
|---------|----------|
| `Ctrl +` | Увеличить |
| `Ctrl -` | Уменьшить |
| `Ctrl+0` | Сбросить до 100% |
| Колесо мыши | Плавное масштабирование |

## Конфигурация

Файл `ocr_tool_config.json` создаётся рядом с `.exe` при первом запуске.

```json
{
  "api_base_url": "https://your-api-server.com",
  "api_key": "your-api-key",
  "ocr_mode": "local"
}
```

| Параметр | Значения | Описание |
|----------|----------|----------|
| `api_base_url` | URL | Адрес OCR API-сервера |
| `api_key` | строка | Ключ аутентификации для API |
| `ocr_mode` | `"local"` / `"api"` | Режим распознавания. `local` — локальные ONNX-модели, `api` — облачный сервер |

> Режим можно переключить прямо в интерфейсе кнопками **Локально** / **API** на боковой панели.
